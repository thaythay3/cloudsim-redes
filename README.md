# ☁️ Simulação de Computação em Nuvem com CloudSim

Projeto desenvolvido para a disciplina de Computação em Nuvem, com o objetivo de compreender o funcionamento de simulações de ambientes em nuvem utilizando o framework CloudSim.

---

## 📚 Sobre o Projeto

O CloudSim é um framework baseado em Java que permite modelar e simular infraestruturas de computação em nuvem.

Neste projeto, foi analisado o exemplo `CloudSimExample1.java`, que demonstra:

- Criação de datacenters
- Configuração de máquinas virtuais (VMs)
- Execução de tarefas (cloudlets)
- Coleta de resultados da simulação

---

## ⚙️ Tecnologias Utilizadas

- Java
- CloudSim
- NetBeans (ambiente sugerido)

---


## ⚠️ Observação

Devido a limitações técnicas do ambiente, não foi possível executar a simulação na prática.  
No entanto, todo o funcionamento foi analisado com base no código e na documentação do CloudSim.
